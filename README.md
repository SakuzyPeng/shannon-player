## 这是什么

香农播放器是一个正在开发中的本地音乐播放器，为珍藏本地音乐文件的人而做——
不追踪、不推流、不打扰，只把你的曲库以舒适的方式呈现出来。

界面依据一套完整的原创设计系统实现，追求安静而耐看的日常质感。

## 特性

- **曲库浏览** —— 专辑网格与详情列表两种视图自由切换
- **悬浮播放条** —— 播放控制、进度、音量、随机与循环一目了然
- **浅色 / 深色 / 跟随系统** —— 一键循环，深色为整套重新调校的配色
- **多语言** —— 简体中文与 English，随时切换
- **顺手的交互** —— 右键菜单支持键盘导航，细节动效温和不喧宾夺主

> 目前处于早期开发阶段：全部界面页面已按设计稿实现，下一步是接入本地曲库
> 扫描与真实音频播放（当前使用种子数据演示）。进展见下方路线图。
