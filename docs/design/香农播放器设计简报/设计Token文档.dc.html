5 · 关键组件约定

  • 封面主色调投影：详情页大封面用 0 26px 68px 14px 的封面主色投影（浅色 ~0.22 透明度、深色 ~0.2）；歌词页封面另有 blur(44px) 主色氛围光（浅 0.30 / 深 0.5）。
  • 爱心三态：未收藏无任何标识 → hover 封面右上浮现操作爱心（唯一交互入口）→ 已收藏在标题右上角标（绝对定位，不占版面、不可点）。列表行内爱心常驻可点。
  • 悬浮播放条：左右 26px、底 22px、高 76px；三段布局 236px / 弹性中列（max 520px）/ 236px；主播放键 44px。
  • 图标栏：宽 84px，按钮 46×44，底部固定"语言（弹出菜单）+ 外观（浅/深/系统三态循环）"。
  • 右键菜单：自绘，222px 宽，role="menu"/"menuitem"，方向键循环 + Enter/Esc，危险项用 --danger。
  • 过滤 vs 搜索：搜索页 = 全局跨类型（范围胶囊：全部/歌曲/专辑/歌手/歌单）；页面内圆钮 = 仅过滤当前列表，命中文字用 --ac 高亮。
  • 列表行网格：44 / 1fr / 170 / 190 / 44 / 60（#、曲名、歌手、专辑、爱心、时长），行高约 40px + 9px 垂直内距。
  • 无障碍：switch 用 role="switch" + aria-checked；菜单键盘导航如上；图标按钮带 title/aria-label；最小可点区域 30px（桌面）。
